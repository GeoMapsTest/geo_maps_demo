This version of the catalog combines five top-level series: AMS and USGS Missouri Topographic Maps, Columbia Map Cabinet, MO Human Geography Cabinet, Missouri Cities Cabinet, and Misc.

About this catalog

This static site reads one merged EAD3 XML finding aid and turns it into a searchable web catalog.

The topographic maps now remain visible as a peer series to the four worksheet-derived cabinets, while nested subseries continue to be searchable at item level.

Data source: docs/data/ead.xml
