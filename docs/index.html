About this Collection

This static site reads one merged EAD3 XML finding aid and turns it into a searchable web catalog.

Data source: docs/data/ead.xml. Digitized by Jingzhou Zhang, SISLT, University of Missouri, Map Scanning Assistant: Andrew Hantak

Search across all five series and click any entry for item details, hierarchy, and scans when available.
