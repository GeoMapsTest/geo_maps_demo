Put scan image files here.

Recommended naming:
- If each item has a single scan:
  itm001.jpg
  itm002.jpg

- If an item has multiple images/pages:
  itm001_1.jpg
  itm001_2.jpg
  ... up to itm001_10.jpg

If you prefer, you can also embed scan links directly in the EAD:
  <dao href="scans/itm001.jpg" role="master"/>

This site will show scans automatically if it can find them.
